粗体、斜体

使用 * 和 表示粗体和斜体。

粗体: **粗体** or 粗体 (快捷键 Command-B)

斜体: * 斜体* or -斜体- (快捷键 Command-I)

标题

使用 === 表示一级标题，使用 — 表示二级标题。

Header 1
========

Header 2
--------

你也可以选择在行首加井号表示不同级别的标题 (H1-H6)，例如：# H1, ## H2, ### H3，#### H4。

# Header 1
## Header 2
### Header 3
#### Header 4
##### Header 5
###### Header 6

链接和电子邮件

内联的

只需在电子邮件周围放置< >即可点击：username@gmail.com
<username@gmail.com>

网址也一样：https://github.com
<https://github.com>

也许你想要一些这样的链接文字：Macdown Website
[Macdown Website](http://macdown.uranusjr.com "Title")（标题是可选的）

参考风格

有时看起来太乱了，不能包含内联的大长网址，或者你想要将所有网址保持在一起。

Make a link [a link][arbitrary_id] then on it’s own line anywhere else in the file:
[arbitrary_id]: http://macdown.uranusjr.com "Title"

If the link text itself would make a good id, you can link like this [like this][], then on it’s own line anywhere else in the file:
[like this]: http://macdown.uranusjr.com 

[arbitrary_id]: http://macdown.uranusjr.com "Title"
[like this]: http://macdown.uranusjr.com

图片 (Image)

内联的

使用 ![描述](图片链接地址) 插入图像。

参考风格

![Alt Image Text][image-id]
on it’s own line elsewhere:
[image-id]: path/or/url/to.jpg "Optional Title"
